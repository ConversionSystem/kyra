## Research notes

Claude Code changed MCP tool loading in April 2026: tool names only load at startup; full schemas are fetched on demand via ToolSearch when a tool fires. Tool descriptions and server instructions are also capped at 2KB to prevent OpenAPI-generated servers from bloating context. Source: Claude Code changelog (code.claude.com/docs/en/changelog) and analysis by alexop.dev.
