## WHAT KYRA IS

**One-liner:** Kyra is a white-label AI workforce platform built on OpenClaw for agencies. Agencies use it to deploy, manage, and monetize autonomous AI workers for their clients — without writing code, without managing infrastructure.

**Category we own:** "AI Workforce Management for Agencies"

**The core insight:** There is a fundamental difference between a chatbot (waits to be triggered, receives input, returns output, reactive, stateless) and an AI worker (operates autonomously, monitors, decides, acts, proactive, persistent). Kyra deploys AI workers. Not chatbots.

**Powered by OpenClaw** — an AI runtime, not a chatbot API. OpenClaw provides: persistent operation (never turns off), full memory (every contact, every channel, forever), tool execution (books appointments, updates CRM, escalates), per-client isolation, model flexibility (Claude, GPT-4, Gemini, BYOK).
